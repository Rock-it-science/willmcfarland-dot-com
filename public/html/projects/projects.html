Discogs Seller Price Comparator (DSPC)


The idea behind this project was to make it simpler to find record sellers on Discogs who had the most records on your wantlist, and compare prices of records including shipping prices.
I managed to make a working version with limited functionality in Java, just through web-scraping, and outputting to a spreadsheet. However, this method would run into errors because of the sheer number of page requests it needed to send.
I attempted another version using the official Discogs php api, however it didn't have support for fetching from wantlists. So I created that method myself in a forked version of the api. Unfortunately, I ran into some other problems that caused me to leave this project for now, but maybe one day I'll come back to it.
